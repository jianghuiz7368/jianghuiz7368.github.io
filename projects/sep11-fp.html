Sep11 Freedom Project

The SEP11 Freedom Project is about using javascript and a tool that you chose to create anything that can be used by others or benefits the society.

For my project, my partner, JunJie Li and I decided to create a typing game. The tool that I chose for this project is firebase and my partner's tool is kaboom.js. I used firebase to create the scoreboard while my partner used kaboom.js to make the game itself. In the process of learning my tool and creating the product, I had many takeaways and faced many challenges. The biggest takeaway from this project is collaboration and communication. At first my partner and I were just on our own doing our own part of the game but when we're close to finishing our project we noticed that it's hard to connect them since we worked on different platforms.
