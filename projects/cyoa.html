Chooose Your Own Adventure Project

The goal of this project is to practice collaborting on a project and practicing linking files to another file. During the project we had to create a choose your own adventure website

My groupmates and I created a choose your own adventure website on github. It has many storylines and different endings.

One challenge I hawas linking some of the file to another file. For example, our project had a file called home.hd tml and I was trying to link another file to home.html, but home.html is one directory above the file that I'm trying to link to it. At first I tried to use the github.io link for the home.html, but it didn't work. I looked at the link for my other file and I realized that they didn't use the github.io link, they just linked by using their folder and file name. After realizing that I went to the file that I was trying to link to home.html and fixed the link. One thing I take away from this project is that you can just use the file and folder name to link and if one of the file uses the github.io link and the other files did not then the link will not work correctly. My next step is to use what I learned during this project on future projects. In future projects I hope there will be better collaboration and the files will be linked better.
